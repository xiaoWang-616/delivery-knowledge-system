# 风险清单

这个文件记录项目级风险。模块内的小问题可以放在模块问题卡片里，跨模块或高影响问题放在这里。

## 1. 风险等级

| 等级 | 含义 | 是否阻塞开发 |
| --- | --- | --- |
| P0 | 不确认会写错核心业务 | 是 |
| P1 | 可以继续开发，但必须记录并后续确认 | 否 |
| P2 | 不影响主流程，测试或视觉阶段处理 | 否 |

## 2. 风险总表

| 编号 | 风险 | 类型 | 影响模块 | 等级 | 责任方 | 状态 | 处理方案 |
| --- | --- | --- | --- | --- | --- | --- | --- |
| R001 |  | 接口/业务/样式/测试/性能 |  | P0 |  | 待确认 |  |

## 3. 接口风险

| 风险 | 接口 | 影响 | 等级 | 处理方式 |
| --- | --- | --- | --- | --- |
|  |  |  |  |  |

常见接口风险：

- 文档没有接口。
- 文档字段和实际返回不一致。
- 前端想要筛选字段，但接口未定义。
- 写操作字段、状态流转、错误码不清楚。

## 4. 样式风险

| 风险 | 页面 | 参考源 | 等级 | 处理方式 |
| --- | --- | --- | --- | --- |
|  |  |  |  |  |

常见样式风险：

- Demo 和旧项目样式不一致。
- 设计图缺少空态、loading、错误态。
- 表格列太多，小屏或窄屏溢出。
- 组件库现有组件无法完全复刻 Demo。

## 5. 业务风险

| 风险 | 模块 | 影响 | 等级 | 处理方式 |
| --- | --- | --- | --- | --- |
|  |  |  |  |  |

常见业务风险：

- 权限边界不清楚。
- 状态机不清楚。
- 删除、停用、审核、金额类操作规则不清楚。
- 依赖其他模块数据，但来源不明确。

## 6. 性能风险

| 风险 | 页面 | 影响 | 等级 | 处理方式 |
| --- | --- | --- | --- | --- |
|  |  |  |  |  |

常见性能风险：

- 列表全量拉取后前端过滤。
- 搜索每输入一个字就请求接口。
- 写操作后整页刷新。
- 多个组件重复请求同一个接口。
- 进入页面请求大量暂时不可见的数据。

| 风险 | 场景 | 影响 | 等级 | 处理方式 |
| --- | --- | --- | --- | --- |
|  |  |  |  |  |

常见测试风险：

- 测试环境缺数据。
- 账号权限不足。
- 接口偶发失败。
- 没有可验证的边界数据。
