# 接口能力矩阵：代理商管理

## 1. 接口能力表

| 功能能力 | 接口 | 方法 | 可靠性 | 前端处理 |
| --- | --- | --- | --- | --- |
| 代理商列表 | `/agent-api/api/v1/admin/agents` | GET | 可联调 | 按契约传 `keyword/cooperationStatus/provinceCode/businessScopeCode/page/pageSize` |
| 区级筛选 | 同列表接口 | GET | 未支持 | 不传 `cityCode/districtCode/regionCode`，显示提示 |
| 新增代理商 | `/agent-api/api/v1/admin/agents` | POST | 已稳定 | 成功后刷新列表 |
| 代理商详情 | `/agent-api/api/v1/admin/agents/{id}` | GET | 已稳定 | 详情 bundle 的主数据 |
| 编辑基础资料 | `/agent-api/api/v1/admin/agents/{id}` | PUT | 已稳定 | 成功后刷新详情和列表 |
| 编辑财务信息 | `/agent-api/api/v1/admin/agents/{id}/finance` | PUT | 已稳定 | 成功后刷新详情和列表 |
| 编辑邮寄信息 | `/agent-api/api/v1/admin/agents/{id}/mailing` | PUT | 已稳定 | 成功后刷新详情和列表 |
| 设为观察 | `/agent-api/api/v1/admin/agents/{id}/observe` | POST | 已稳定 | 需要原因 |
| 恢复活跃 | `/agent-api/api/v1/admin/agents/{id}/activate` | POST | 已稳定 | 需要原因 |
| 停用代理商 | `/agent-api/api/v1/admin/agents/{id}/disable` | POST | 已稳定 | 后端阻断跟进中商机 |
| 授权区域列表 | `/agent-api/api/v1/admin/agent-authorizations/{agent_id}` | GET | 已稳定 | 详情区块 |
| 新增授权区域 | `/agent-api/api/v1/admin/agent-authorizations/{agent_id}` | POST | 已稳定 | 请求用 `regionCode/regionLevel` |
| 编辑授权区域 | `/agent-api/api/v1/admin/agent-authorizations/{agent_id}/{id}` | PUT | 已稳定 | 请求用 `regionCode/regionLevel` |
| 停用授权区域 | `/agent-api/api/v1/admin/agent-authorizations/{agent_id}/{id}/disable` | POST | 已稳定 | 需要原因 |
| 协议列表 | `/agent-api/api/v1/admin/agent-agreements/{agent_id}` | GET | 已稳定 | 详情区块 |
| 新增协议 | `/agent-api/api/v1/admin/agent-agreements/{agent_id}` | POST | 已稳定 | 先走 common 文件上传 |
| 作废协议 | `/agent-api/api/v1/admin/agent-agreements/{agent_id}/{id}/void` | POST | 已稳定 | 需要原因 |
| 文件对象 | `/common-api/api/v1/admin/files/{fileId}` | GET | 可联调 | 用于协议下载 |
| 推广目标 | `/agent-api/api/v1/admin/agent-promotion-targets/{agent_id}` | GET/POST/PUT/DELETE | 已稳定 | 详情区块 |
| 成员 | `/agent-api/api/v1/admin/agent-members/{agent_id}` | GET/POST/PUT | 已稳定 | 销售/运营 tab |
| 成员启停 | `/agent-api/api/v1/admin/agent-members/{agent_id}/{id}/enable/disable` | POST | 已稳定 | 需要原因 |

## 2. 参数禁止清单

| 参数 | 原因 | 替代方案 |
| --- | --- | --- |
| `cityCode` | 代理商列表接口未定义 | 只传 `provinceCode`，提示区级未支持 |
| `districtCode` | 代理商列表接口未定义 | 只传 `provinceCode`，提示区级未支持 |
| `regionCode` | 代理商列表接口未定义 | 授权区域接口可用，列表不传 |

## 3. 字段不确定清单

| 字段 | 不确定点 | 当前处理 |
| --- | --- | --- |
| `subjectType` | 中文映射未明确 | 原样展示或 `-` |
| `taxpayerType` | 枚举未明确 | 原样展示或 `-` |
| `vatTaxRate` | 是比例还是百分比未明确 | 原样展示 |
| 协议状态 | 状态枚举不完整 | `5` 展示已作废，其余展示生效中 |

## 4. 降级策略

```text
区级筛选：
- 允许用户选择到区。
- 不传接口未定义的 cityCode/districtCode/regionCode。
- 前端提示“当前接口不支持市/区精确筛选，按省份查询”。

协议下载：
- 通过 fileId 获取文件对象。
- 如果没有 downloadUrl，提示下载地址失效。
- 等待真实协议数据做最终点测。
```
